# How to Read a Scientific Paper

A neat guide about how to properly read a scientific paper. [https://www.sciencemag.org/careers/2016/03/how-seriously-read-scientific-paper](https://www.sciencemag.org/careers/2016/03/how-seriously-read-scientific-paper)

Summary:

* Read Abstract First
* Glance through Figures and Charts
* Read Conclusions
* Read Paper from start to end
